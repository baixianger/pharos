How it works

The core loop

Four steps from zero to agent — then back around, endlessly.

01
👁
See your projects

Watchlist sidebar groups all your repos with live git status and commit activity at a glance.

02
⚡
Launch an agent

One ⌘K command fires Claude Code or Codex into any repo — yolo, tmux, or your favourite terminal & editor.

03
↺
Resume a session

Full session browser with timestamps. One click puts you right back where the agent left off.

04
⫶
Run them in parallel

Git worktree manager spins up isolated branches so multiple agents never step on each other.
