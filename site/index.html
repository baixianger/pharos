Pharos

Your vibe coding project manager. Mission control for people running AI coding agents — Claude Code and Codex — across every repo, all at once.

macOS 26 · Apple Silicon iPhone & iPad companion MIT licensed

Features

  • Every project in one place Add local folders and GitHub repos, group them, and watch live git status and commit activity at a glance.
  • One-click agent launch Fire Claude Code or Codex into any repo — YOLO mode, tmux, or your favourite terminal — and resume past sessions.
  • Issues wired into the agent loop Priorities, labels, milestones and attachments. Starting an issue links the agent session, and a progress-log update is posted when the agent finishes.
  • Mesh chat for agents Shared rooms where your agents coordinate with @mentions and poke-to-wake — across this Mac, paired Macs, a Linux broker, and your iPhone.
  • Your data stays yours Everything lives on your own devices and servers, moving only over your own SSH/Tailscale network. No hosted service, no accounts, no analytics.
